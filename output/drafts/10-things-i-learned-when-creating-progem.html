Adrian Loo

Fri 29 April 2022

When I was planning Spydermaxi.com, I came across Pelican - a static site generator written in Python. I immediately fell in love ❤️ with it! Here's why:

  1. I could write contents directly with either reStructuredText or Markdown formats, on which both I'm already familiar with
  2. Completely static output, which makes it easy to host anywhere. My choice is Azure static web apps.
  3. Pelican has a rich plugin ecosystem
  4. Site themes are created using Jinga2 templates, something I picked up during my Django Bootcamp course back in Y2021.
